but hard to analyze at scale

Transcript Parsing

// Raw transcript
"Tim Cook -- CEO:
  We're thrilled to report record revenue
  of $94.8 billion for the quarter..."

// Parsed chunk
{
  chunk_id: "AAPL_2024Q3_0012",
  speaker: "Tim Cook",
  role: "CEO",
  section: "prepared_remarks",
  text: "We're thrilled to..."
}

Processing Steps

1. Boilerplate Removal

Strips safe harbor disclaimers, copyright notices, legal text

2. Speaker Detection

6 regex patterns match formats like Name -- Title, Name (Title):

3. Role Classification

Maps to CEO, CFO, Analyst, Operator, etc. using title keywords

4. Section Detection

Splits into Prepared Remarks and Q&A

5. Sentence-Aware Chunking

~400 tokens per chunk, 50-token overlap, respects sentence boundaries

NLP Analysis

Three-method sentiment analysis

📚

Loughran-McDonald

Finance-specific lexicon. Unlike general dictionaries, correctly handles financial language.

Score Formula
net = (pos - neg) / total
🤖

VADER

Rule-based model that handles punctuation, capitalization, degree modifiers, and negation.

"Great"+0.51
"Not great"-0.30

FinBERT

Pre-trained BERT fine-tuned on financial text. Deep contextual understanding.

Context-Aware Processing

// Negation detection (3-word window)
"We see no margin pressure"
  ➔ negated = true ➔ excluded

// Severity scoring (5-word window)
"significant supply constraints"
  ➔ severity = "high" (weight: 3)

"minor supply constraints"
  ➔ severity = "low" (weight: 1)

Output per chunk:

risk_categories risk_count risk_intensity risk_details

Retrieval-Augmented Generation

Hybrid search + local LLM

🔍

1. Dense Search (FAISS)

all-MiniLM-L6-v2 embeddings, 384-dim vectors, cosine similarity
"revenue growth" matches "top-line expansion"

+
📝

2. Sparse Search (BM25)

Token-level keyword matching with TF-IDF weighting
Exact match for "EBITDA", "EPS", specific figures

↓
🔀

3. Reciprocal Rank Fusion

score = 0.6/(k+rankdense) + 0.4/(k+ranksparse)
Best of both worlds — semantic + exact

↓
🤖

4. Llama 3.2 (Local via Ollama)

Free, no API key. Financial analyst system prompt.
Generates cited answers from retrieved context
